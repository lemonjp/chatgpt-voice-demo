## Setup

Install `ffmpeg` with:
```
brew install ffmpeg
```

Install `openai`, `gradio` with:
```
pip3 install -r requirements.txt
```

or you have a path to python3

```
pip install -r requirements.txt
```

## Lounch

```
python3 web.py
```

then go to `http://127.0.0.1:7860`
